== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Zoom IMG'
3. Activate Zoom IMG from your Plugins page.

=== Manually ===

1. Upload the `zoom-img` folder to the `/wp-content/plugins/` directory
2. Activate the Zoom IMG plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
